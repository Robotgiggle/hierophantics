      Активаторы Иерофанта

      Мои встроенные разумы самостоятельны лишь до некоторой степени — они не могут по-настоящему мыслить. Поэтому для начала сотворения заговора им нужен какой-то активатор. Паттерны, перечисленные здесь, используются для создания таких активаторов на основе различных часто встречающихся событий. Созданный активатор можно назначить разуму с помощью Гамбита активатора.


      Некоторые активаторы при срабатывании могут помещать на стек йоту, связанную с событием, что позволяет более тонко реагировать. Активаторы, которые так делают, перечислены ниже вместе с йотой, которую они предоставляют:

      Урон: величина урона

      Тип урона: величина урона

      Сброс: сущность выброшенного предмета

      Атака: атакованная сущность

      Разрушение: позиция разрушенного блока

      Телепортация: вектор смещения

                                                          Шаблоны Иерофанта

                                                          Когда я встрою в себя разум, мне нужно будет указать ему его цель. Это делается с помощью йот-ссылок на разум, которые работают так же, как йоты сущностей, но для встроенных разумов. Таким образом, я могу свободно создавать и уничтожать эти йоты, не влияя на существование самих разумов.


                                                          Заметка о приватности

                                                          Поскольку ссылки на разум можно записывать на носители, другой игрок может получить доступ к йотам, ссылающимся на мои разумы. Такой человек не сможет изменять мои разумы (результат будет подобен попытке сохранить моё Истинное Имя), но он сможет просмотреть активатор и заговор, связанные с каждым разумом.
